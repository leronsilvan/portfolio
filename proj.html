Car Price Predictor

Car Price Predictor


Predicts the resale price of a car based on specifications such as brand, year, fuel type, and mileage.
Uses Linear Regression after proper encoding and preprocessing.

• Built with: Python, Streamlit, scikit-learn, pandas
• Input: Car brand, year, fuel type, kilometers driven, etc.
• Output: Estimated resale price
• Use Case: Helps users get fair resale value for their vehicles.

Stock Price Predictor

Stock Price Predictor


Forecasts the next day’s stock price using a sliding window approach with Linear Regression.

• Built with: Python, Streamlit, yfinance, scikit-learn
• Input: Stock symbol and date range
• Output: Predicted closing price
• Use Case: Helps users make informed trading decisions.

Loan Approval Classifier

Loan Approval Classifier


Predicts whether a loan application will be approved based on applicant financial and background data.
Implements a classification model trained on real-world-like features.

• Built with: Python, Streamlit, scikit-learn
• Input: Income, CIBIL score, employment, loan amount, assets
• Output: Approved / Rejected
• Use Case: Early screening for loan eligibility.

Scholarship Eligibility

Scholarship Eligibility Classifier


Predicts whether a student is eligible for a scholarship based on personal and academic features.
It uses K-Nearest Neighbors (KNN) for classification.

• Built with: Python, Streamlit, scikit-learn
• Features: Categorical + numerical student data
• Output: Eligible or Not Eligible
• Use Case: Helps in pre-screening scholarship applicants.

Audio Genre Predictor

Audio Genre Predictor


This project classifies songs into genres based on extracted audio features using unsupervised learning (KMeans Clustering).

• Built with: Python, Streamlit, librosa, scikit-learn
• Input: .mp3 or .wav audio file
• Output: Predicted genre cluster
• Use Case: Helps categorize music automatically using audio characteristics like tempo, rhythm, and spectral features.
